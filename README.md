# Teacher Checklist Block

O Teacher Checklist é um bloco para Moodle que ajuda professores a acompanharem o progresso da configuração de seus cursos.

## Funcionalidades
* **Detecção Automática:** Verifica se o curso está visível, se há atividades sem nota, fóruns vazios, etc.
* **Itens Manuais:** O professor pode adicionar suas próprias tarefas (ex: "Imprimir provas").
* **Backup Seguro:** Preserva as anotações manuais do professor ao restaurar o curso.
* **Conformidade LGPD:** Suporte total à API de Privacidade.

## Instalação
1. Copie a pasta `teacher_checklist` para o diretório `blocks/` do seu Moodle.
2. Acesse a administração do site para instalar.
